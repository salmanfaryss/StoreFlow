# 🛒 StoreFlow

StoreFlow, ASP.NET Core MVC, Entity Framework Core ve SQL Server kullanılarak geliştirilmiş modern bir envanter ve sipariş yönetim sistemidir.

Bu proje sayesinde ürünler, kategoriler, müşteriler ve siparişler merkezi bir panel üzerinden yönetilebilmektedir. Ayrıca dashboard ekranı üzerinden çeşitli istatistikler ve raporlamalar görüntülenebilmektedir.

## 🚀 Kullanılan Teknolojiler

* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* LINQ
* HTML5
* CSS3
* Bootstrap
* JavaScript

## 📌 Proje Özellikleri

### Kategori Yönetimi

* Kategori ekleme
* Kategori güncelleme
* Kategori silme
* Kategori listeleme

### Ürün Yönetimi

* Ürün ekleme
* Ürün güncelleme
* Ürün silme
* Ürün listeleme
* Stok takibi

### Müşteri Yönetimi

* Müşteri kayıtları
* Müşteri bilgilerini görüntüleme
* Müşteri güncelleme işlemleri

### Sipariş Yönetimi

* Sipariş oluşturma
* Sipariş takibi
* Sipariş durum yönetimi
* Toplam tutar hesaplama

### Dashboard

* Toplam ürün sayısı
* Toplam kategori sayısı
* Toplam müşteri sayısı
* Toplam sipariş sayısı
* Çeşitli istatistiksel raporlar

### Diğer Modüller

* Activity Yönetimi
* Message Yönetimi
* Todo Yönetimi

## 📊 Bu Projede Öğrendiklerim

Bu proje boyunca özellikle aşağıdaki konularda önemli deneyimler kazandım:

* 70+ LINQ sorgusu
* Where sorguları
* Select işlemleri
* OrderBy ve OrderByDescending
* FirstOrDefault kullanımı
* Count, Sum, Average, Min, Max işlemleri
* Group By sorguları
* Join işlemleri
* İlişkili tablolarla çalışma
* Entity Framework Core CRUD işlemleri
* Dashboard istatistikleri oluşturma
* SQL Server veri yönetimi

## 🎯 Amaç

Bu proje, ASP.NET Core MVC ve Entity Framework Core konularında edinilen bilgilerin gerçek bir iş senaryosu üzerinde uygulanması amacıyla geliştirilmiştir.
